# SLADE Synthetic New Dataset

Synthetic dataset for testing Type 2 & 3 anomalies (new account anomalies).

## Source
- Generated by: [SLADE Repository](https://github.com/jhsk777/SLADE)

## Format
CSV file with columns:
- `u`: Source node ID
- `i`: Destination node ID
- `ts`: Timestamp
- `label`: Anomaly label (0=normal, 1=anomaly)
- `idx`: Edge index

## Statistics
- ~333,200 edges
- Synthetic temporal graph with new account anomalies

## Anomaly Types
- Type 2: New malicious accounts
- Type 3: Mixed anomaly patterns

## Usage
```bash
graflag benchmark -m slade -d slade_synthetic_new --build
```
