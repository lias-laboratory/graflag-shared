# DARPA Dataset (AnoGraph Format)

Network intrusion detection dataset from KDD Cup 1999.

## Format
- `Data.csv`: Edge stream with columns `src,dst,timestamp`
- `Label.csv`: Binary labels (0=normal, 1=anomaly)

## Statistics
- ~4.9M edges
- Network traffic with attack patterns

## Compatible Methods
- anograph (all variants: anograph, anographk, anoedgeg, anoedgel)

https://github.com/Stream-AD/AnoGraph
