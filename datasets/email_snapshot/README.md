# Email Dataset (Snapshot Format)

Email communication network from StrGNN paper.

## Format
- `acc_email.npy`: Accumulated adjacency matrices (T, N, N)
- `split.npz`: Train/test split with edge indices and snapshot IDs

## Compatible Methods
- strgnn
- addgraph

## Source
https://github.com/KnowledgeDiscovery/StrGNN
