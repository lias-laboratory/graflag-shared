# ISCX-IDS2012 Dataset (AnoGraph Format)

Network intrusion detection dataset from ISCX.

## Format
- `Data.csv`: Edge stream with columns `src,dst,timestamp`
- `Label.csv`: Binary labels (0=normal, 1=anomaly)

## Compatible Methods
- anograph (all variants: anograph, anographk, anoedgeg, anoedgel)
