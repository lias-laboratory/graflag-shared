# SLADE Synthetic Hijack Dataset

Synthetic dataset for testing Type 1 & 3 anomalies (account hijacking scenarios).

## Source
- Generated by: [SLADE Repository](https://github.com/jhsk777/SLADE)

## Format
CSV file with columns:
- `u`: Source node ID
- `i`: Destination node ID
- `ts`: Timestamp
- `label`: Anomaly label (0=normal, 1=anomaly)
- `idx`: Edge index

## Statistics
- ~333,200 edges
- Synthetic temporal graph with injected hijacking anomalies

## Anomaly Types
- Type 1: Account hijacking - sudden behavior change
- Type 3: Mixed anomaly patterns

## Usage
```bash
graflag benchmark -m slade -d slade_synthetic_hijack --build
```
